# Student Enrollment System

A user-friendly **Student Enrollment System** built using **Bash scripting** and **Zenity** for graphical interface, combined with Python for email notifications. This project allows administrators to enroll students in various courses, manage student data, and view students enrolled in specific departments.

---

## Features

1. **Login System**
2. **Student Enrollment**
3. **Department-Wise Display**
4. **Courses Offered**
5. **Email Notification**
---

## Technologies Used

- **Bash Scripting**: Backend logic for enrollment and data handling.
- **Zenity**: GUI dialog for user interaction.
- **Python**: SMTP functionality to send email notifications.
- **Environment Variables**: Securely store sensitive data like email credentials.
